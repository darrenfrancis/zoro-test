## Requirements

-   Node version 8 or higher

## Quick start

Choose one of the following options:

1. Clone the repo from [here](https://github.com/darrenfrancis/zoro-test)

2. Install the dependencies (`npm install`)

3. Run the tests with ['npm run wdio']
